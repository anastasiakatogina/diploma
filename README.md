# diploma
ВКР "Сценарное статистическое имитационное моделирование кредитных рисков"
В репозитории находятся материалы ВКР, посвященной имитационному моделированию кредитного риска с использованием метода Монте-Карло.
Структура:

data/ — исходные и обработанные данные
notebooks/ — Jupyter Notebook с реализацией модели
results/ — результаты моделирования

Оценка кредитного риска проводится на основе имитационного моделирования методом Монте-Карло. В рамках модели:

Формируется набор случайных макроэкономических факторов
Учитываются различия между экономическими режимами
Моделируется распределение задолженности
На основе полученного распределения анализируются характеристики риска

Для снижения размерности и устранения мультиколлинеарности используется метод главных компонент (PCA), а для выделения режимов — кластеризация KMeans.

Модель построена на основе открытых статистических данных:

Банк России — показатели ипотечного кредитования
Росстат — данные о доходах населения
ЕМИСС — показатели рынка недвижимости

Исходные данные приведены к единому формату, агрегированы по месяцам и преобразованы в итоговый набор признаков, используемый в моделировании.

В результате моделирования получено распределение возможных значений кредитного риска, позволяющее:

Оценивать потенциальные потери
Анализировать поведение риска в различных сценариях
Выявлять факторы, влияющие на устойчивость кредитования
